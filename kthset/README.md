# 説明
小さい方からK番目の値を高速に取り出すことのできるsetです。
# 使用方法
使用する型と何番目を取り出すかのKを用いて宣言します。
## 例
int型で、5番目を取り出すset
```cpp
Kthset<int> st(5);
```
string型で、10番目を取り出すset
```cpp
Kthset<string> st(10);
```
## メンバ関数の説明
* `insert(v)`
値`v`を追加します。
元々`v`が存在していた場合追加されません。
* `erase(v)`
値`v`を削除します。
* `size()`
要素数を取得します。
* `getKth()`
小さい方からK番目の値を取得します。
要素数がK未満の場合はエラーが出ます。
